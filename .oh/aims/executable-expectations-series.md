# Aim: Strategy Reversal and Executable Memory

## Desired reader outcome

Readers should understand apparent strategy dysfunction as a media reversal, not just leadership failure or execution sloppiness. The essay should distinguish declared strategy from operative strategy; define organizational media as recurring artifacts plus workflow, authority, and incentives; show how those media reverse when representations can be satisfied while underlying objectives worsen; and close on consequence-bearing memory that changes the next action.
