# Metis: Change Lineage and Executable Memory

## Do not split where the subject changes

The failed draft architecture treated adjacent conceptual subjects as publication units: organizational communication, dissent, reversal, process variation, and the Recon Agent harness. That preserved local readability while destroying the dependencies that made the argument forceful.

Consequence: split only after a completed transformation: original function → enhancement → saturation → reversal → new necessity. If a section's later payoff lives across the boundary, the cut is probably wrong.

## A canonical spine is an editorial control surface

The body of work needs a durable argument graph, not just drafts. Future writing agents can preserve subject matter while destroying conceptual architecture if they do not know which claims depend on which prior movement.

Consequence: `.oh/sources/SERIES_SPINE.md` is load-bearing. It records the governing thesis, causal sequence, recurring example, protected hinge sentences, and editorial expectations. Future rewrites must update the spine when they intentionally change dependencies.

## The solution must arrive late

Archaeology and executable expectations sound like knowledge management or workflow governance if they appear before the failure of documentation, decision logs, governance, and AI summaries has been established.

Consequence: the canonical essay must make the reader feel that content-level remedies remain inside the broken medium before presenting harvested lineage and executable precedent.

## Dissent is a state-changing interface, not a separate content lane

The standalone dissent draft was locally useful but structurally fragmenting. Dissent belongs inside the communication reversal argument as a mechanism: evidence from the work must be able to alter assumptions, scope, funding, ownership, continuation, or authority.

Consequence: do not create a standalone dissent post unless it carries a complete transformation and the flagship remains intact. Otherwise use dissent as work-to-strategy feedback inside the canonical essay's diagnostic or constructive movement.

## Quotes need a ledger

A research pass creates new hallucination risk. It is not enough to add plausible names and links. Every public quote must be traceable to text read in the session or already present in a verified local source.

Consequence: `.oh/sources/research-quotes-ledger.md` records exact quotes, source URLs or local paths, and no-use constraints. Potter is not directly quoted in new prose because the primary source was not accessible in this environment.

## The concrete example is the continuity mechanism

The reconciliation failure is not merely an example for the constructive piece. It is the recurring case that carries the abstract argument through the arc: evidence, ceremonial receipt, unchanged decision, LLM-mediated mutation, lost lineage, archaeological reconstruction, executable expectation, and named override.

Consequence: future edits should keep the chronology and actors stable. Changing the example requires updating the spine and every payoff that depends on it.
